# PowerBI-Loading-transformed-data

PowerBI exercises were perfomed online in Skillable Labs.
In this lab we completed the following in the Power Query Editor window:
1. renaming queries
2. selecting columns
3. sorting and filtering columns
4. renaming, merging and removing columns
5. using 'Applied steps'
6. expanding tables by selecting additional columns from a related table
7. replacing values
8. creating a custom column
9. configuring the data type
